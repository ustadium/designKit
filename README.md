# uSTADIUM Design Kit

A lightweight Swift package that provides reusable UI components, including a customizable bottom alert view and an OTP input view. Designed for SwiftUI projects, this pod helps reduce repetitive code and speed up common UI workflows.

---

## Features
- Customizable bottom alert component  
- Liquid-glass style visual effects (optional)  
- Reusable SwiftUI OTP input field with auto-fill support  
- Clean, modular architecture  
- Easy integration into any SwiftUI project  

---

## Installation

### CocoaPods

To install **YourPodName** using CocoaPods, add the following line to your `Podfile`:

pod 'UstadiumDesignKit'
